## 로딩에 대한 고민

토이 프로젝트를 진행하던 중 로딩 상태에 대한 고민이 있었다. 데이터의 로딩 상태를 progress bar 로 표현해줄 수 있었지만, Skeleton UI 를 적용해보고 싶었다.

### 지금까지의 로딩처리

지금까지는 렌더링에 필요한 데이터가 없을때 “Loading…” 이라는 문구만 표시해주었는데, 이게 완성도가 많이 떨어져 보였다.

### Skeleton UI 만들어보기

진행중인 토이프로젝트가 화면이 하나이고 모든 요소가 데이터가 필요한 요소들이었기에 모든 요소들을 하나의 컴포넌트로 만들어 스켈레톤 UI 를 만들었다.

### 더 잘 만들수 있을까 ?

컴포넌트에 boolean 형의 loading을 전달하는 것으로 컴포넌트 자체에 skeleton ui 를 구현해두고 true 일땐 스켈레톤 ui 를 false일땐 원래의 컴포넌트로 렌더링하는게 좋을 것 같다.
