## 상태 추가하기

```jsx
const [passwordType, setPasswordType] = useState({
  type: "password",
  visible: false,
});
```

비밀번호의 상태와 타입을 저장하기 위해 `useState` 를 사용했다. `type` 에서는 비밀번호의 타입을 지정해주고, `visible` 에서는 비밀번호를 볼 수 있는 상태를 지정해주었다.

## 타입을 바꿔주는 함수 만들기

```jsx
const handlePasswordType = () => {
  setPasswordType(() => {
    if (!passwordType.visible) {
      return { type: "text", visible: true };
    } else {
      return { type: "password", visible: false };
    }
  });
};
```

`passwordType.visible` 타입이

- true 일때 `type: "text", visible: true` 를 리턴하여 `input`의 타입에는 text를 버튼의 상태에는 true를 전달한다.
- false 일때는 `type: "password", visible: false` 를 리턴하여 input의 타입에는 password를 버튼의 상태에는 false를 전달한다.

## Input 컴포넌트

```jsx
<Input
  label="비밀번호"
  placeholder="비밀번호를 입력해주세요."
  Type={passwordType.type}
  icon={passwordType.visible ? <HiOutlineEye /> : <HiOutlineEyeSlash />}
  handleIconClick={handlePasswordType}
/>
```

아이콘이 클릭되었을때, 위에 작성한 함수를 불러오고, `type` 과 `icon`에서 `passwordType` 의 정보를 불러와서 비밀번호 컴포넌트를 만들수 있었다.
